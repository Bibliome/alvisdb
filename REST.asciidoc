= API

== Entity types

---
/entities
---

Return all known entity types.

=== Parameters

|===
| Parameter | Source | Type | Effect

| `fields` | Query | `String[]` | Information to display for each entity type `id` `name` `root` `all`
|===

=== Examples


[source,json]
.http://bibliome.jouy.inra.fr/demo/alvisdb/obt/entities[`/entities`]
----
[
    {
        "id": "habitat",
        "name": "Habitat",
        "root": "OBT:000000" <1>
    },
    {
        "id": "taxon",
        "name": "Taxon",
        "root": "2"
    }
]
----
